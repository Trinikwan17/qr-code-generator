# 🔳 QR Code Generator

A modern and responsive QR Code Generator built using React and Vite.  
This application allows users to generate and download QR codes instantly from text or URLs.

---

## 📌 Features

- ⚡ Instant QR code generation
- 🔗 Supports text and URL input
- 🎨 Clean and responsive UI
- 🔄 Generate multiple QR codes dynamically
- 📱 Mobile-friendly design
- ⬇️ Download generated QR code as image
- ⚛️ Built with React + Vite

---

## 🛠 Tech Stack

- React
- Vite
- JavaScrip
- CSS3
- QR Code generation library

---

## 🧠 How It Works

- User enters text or a URL.
- On clicking generate, a QR code is created dynamically.
- The QR updates instantly when the input changes.
- The generated QR can be scanned using any QR scanner.
- User can download the QR code as image.

## 🚀 Installation & Setup

1️⃣ Clone the repository

```
git clone https://github.com/Trinikwan17/qr-code-generator.git
```

2️⃣ Navigate into project folder

```
cd qr-code-generator/my-app
```

3️⃣ Install dependencies

```
npm install
```

4️⃣ Start development server

```
npm run dev
```
